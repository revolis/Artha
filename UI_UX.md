# UI/UX Direction

## Theme
- Light theme only (premium, polished)
- Warm neutral background with subtle surface contrast
- Private Mode masks values everywhere

## Typography
- Headings: Apple-style system stack with tighter tracking
- Body: System stack (-apple-system, system-ui) for native feel
- Avoid default system stacks

## Color Palette (Suggested)
- Background: #F6F3EF
- Surface: #FFFFFF
- Surface muted: #EEE7DF
- Text primary: #1E1C18
- Text secondary: #6D675F
- Accent: #2F6B5E
- Accent soft: #CFE2DD
- Positive: #1B6F5A
- Negative: #B04A3A
- Warning: #C48B2A

## Layout and Density
- Generous spacing and breathing room
- Card-based sections with subtle elevation
- Tables with soft dividers and sticky headers
- Consistent 8px spacing scale

## Data Display
- KPI cards: large value, small label, minimal borders
- Charts: low-contrast gridlines, bold primary series
- Heatmap: auto-scaled intensity, clear legend

## Private Mode
- Replace all amounts with dots (e.g., "••••")
- Hide tooltip numeric values
- Preserve chart shapes without numbers

## Motion
- Page-load fade and staggered card reveals
- 150-250ms transitions with ease-out
- Avoid excessive micro-animations
